隐私政策

一、信息收集

小工具箱所有工具均在您的浏览器本地运行。我们不会收集、存储、传输您输入的任何数据。所有数据处理均在本地完成。

二、第三方服务

我们使用以下第三方服务：

  • Tailwind CSS（CDN） — 用于页面样式
  • Lucide Icons（CDN） — 图标库
  • QRCode.js（CDN） — 二维码生成
  • Google Fonts — 字体服务
  • Buy Me a Coffee — 捐赠收款

三、访问日志

GitHub Pages 托管平台可能会自动记录部分访问日志，包括访问时间、IP（匿名化）和访问页面。这些日志由 GitHub 管理，不受我们控制。

四、Cookie

本工具站不使用任何 Cookie 或追踪技术。

五、联系我们

如对本隐私政策有疑问，请通过 GitHub Issues 或邮箱联系我们。

六、更新

我们可能会不时更新本政策，更新后版本自发布之日起生效。

最后更新：2025年
